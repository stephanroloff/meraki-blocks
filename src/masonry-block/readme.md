How to use it

1 - Create a ACF Field Group called "Masonry Galleries"

2 - Then create a Repeater:

- Label Name: Images
  Field Name: images
  Type: Repeater

3 - Then create the subfields (inside the repeater):

- Label Name: Image
  Field Name: image
  Type: image

- Label Name: Name
  Field Name: name
  Type: text

- Label Name: Description
  Field Name: description
  Type: text

- Label Name: Link (Slug)
  Field Name: link
  Type: text
